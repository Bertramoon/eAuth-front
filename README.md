# eAuth-front
RBAC-based identity authentication and access control system [vue-based front-end part]

The back-end link: [https://github.com/Bertramoon/eAuth](https://github.com/Bertramoon/eAuth )

## Setup

```sh
npm install
```

## Compile and Run

### Development

```sh
npm run dev
```

### Production

```sh
npm run prod
```

## Build

```sh
npm run build
```
